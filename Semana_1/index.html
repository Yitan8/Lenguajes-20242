Etiquetas de cabeza de documento:

<head></head>

Es la primera sección de un documento HTML se ubica la cabeza haciendo referencia a un cuerpo. La Cabeza contiene información que no será mostrada en el navegador, como lo es el título, enlace al documento CSS, el idioma de la página, información del autor, el ícono de la pestaña.

<meta>

en conjunto con los atributos muestran los metadatos de la página como lo es la información de codificación de caracteres(charset="utf-8"), nombre del tipo de contenido que tendrá el archivo (name="") y su contenido o descripción(content=""). también se puede incluir el ícono de la página, está se visualizará al lado del URL y al momento de guardar la página como favoritos.

<title>'Aquí va el titulo del documento'</title>

agrega un título a todo el documento HTML. Este será visto en la pestaña y al momento de guardar la página como favoritos.

<link rel="(stylesheet,icon...)" href="(ruta relativa)" />

etiqueta usada para enlazar recursos externos al documento actual. El atributo "href" sirve para indicar la ruta relativa.

Etiqueta no semántica, de preferencia se trabaja en conjunto con el atributo class= '' se usa para darle reglas visuales al texto contenido al momento de señalarlo en CSS o JS.

</ul>

<ol>

Etiqueta usada para marcar un fragmento de texto para resaltarlo semánticamente y hacerlo más atractivo para el lector. Se mostrará como cursiva.

<p>En ciertas palabras, frases o ideas se usa la negrita para dar <strong>'una mayor importancia'</strong> a términos específicos dentro de él</p>

En ciertas palabras, frases o ideas se usa la negrita para dar una mayor importancia a términos específicos dentro de él

Etiqueta usada para marcar un texto como importante. Normalmente se mostrará el texto como negrita.
